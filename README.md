# World Clock Dashboard

A real-time world clock dashboard showing time across multiple countries with beautiful analog and digital displays.

## Test Message for Git practice

## Features

- Real-time updates
- Analog wall clock design
- Digital time display
- Date display
- Supports multiple time zones:
  - India (New Delhi)
  - USA (Central Time)
  - Malaysia (Kuala Lumpur)
  - Thailand (Bangkok)
  - Russia (Moscow)

## Live Demo

Visit the live website: [World Clock Dashboard](https://vijayainf.github.io/vk-world-clock)

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Setup

1. Clone the repository
2. Open `index.html` in a web browser
